# Privacy Screen — Smoke Tests

Ручные сценарии проверки для релизного чеклиста.

## Android

### Тест 1: Recent Apps — карточка скрыта
1. Настройки → «Экран конфиденциальности» → включить
2. Нажать кнопку «Обзор задач» (квадрат/жест снизу вверх)
3. **Ожидаемо:** карточка приложения серая или пустая, контент не виден
4. **Фактически:** [записать результат]

### Тест 2: Скриншот заблокирован
1. Убедиться, что флаг включён
2. Попытаться сделать скриншот (Power + Vol Down или жест)
3. **Ожидаемо:** уведомление «Не удалось сделать снимок экрана» или тёмное изображение
4. **Фактически:** [записать результат]

### Тест 3: Отключение флага
1. Выключить «Экран конфиденциальности»
2. Открыть Recent Apps
3. **Ожидаемо:** карточка показывает контент приложения
4. **Фактически:** [записать результат]

## iOS

### Тест 1: App Switcher — blur overlay
1. Настройки → «Экран конфиденциальности» → включить
2. Свернуть приложение (жест вверх или двойной Home)
3. В app switcher карточка приложения должна показывать blur overlay (тёмный экран с иконкой замка)
4. **Ожидаемо:** контент скрыт, виден только overlay
5. **Фактически:** [записать результат]

### Тест 2: Возврат из background
1. Вернуться в приложение из app switcher
2. **Ожидаемо:** overlay исчезает, контент доступен
3. **Фактически:** [записать результат]

### Тест 3: Screen Recording (iOS 11+)

2. **Ожидаемо:** overlay появляется поверх контента приложения
3. **Ограничение:** запись продолжается, но контент скрыт overlay
4. **Фактически:** [записать результат]

## Известные платформенные ограничения

| Ограничение | Android | iOS | Desktop |
|---|---|---|---|
| OS screenshot hotkeys | Заблокированы (FLAG_SECURE) | Не заблокированы | Не заблокированы |
| Screen recording | Заблокирована (FLAG_SECURE) | Обнаруживается, контент скрыт overlay | Не обнаруживается |
| App switcher protection | Полная (серая карточка) | Blur overlay | N/A (нет switcher) |
| Сторонние скриншотеры | Заблокированы | Не заблокированы | Не заблокированы |

Ограничения Desktop задокументированы в `docs/privacy-screen-desktop-limitations.md`.
