# Shared

Общий слой для будущих нативных клиентов.

Здесь живут только platform-neutral контракты и артефакты, которые должны одинаково интерпретироваться `Desktop`, `Android`, `iOS`.

Подкаталоги:

- `protocol/` — REST/WS-контракты и схемы payload
- `domain/` — общие доменные модели
- `crypto-contracts/` — контракты crypto-слоя без platform-specific реализации
- `test-vectors/` — кроссплатформенные тестовые векторы для проверки совместимости
