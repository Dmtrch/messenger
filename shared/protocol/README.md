# Shared Protocol

Пакет для общих протокольных определений.

Назначение:

- фиксировать REST DTO;
- фиксировать WebSocket frame/payload contracts;
- сохранять совместимость между web-клиентом и будущими нативными клиентами.

Ограничения:

- без platform-specific кода;
- без runtime-specific сериализации;
- любые несовместимые изменения только через отдельный ADR или RFC.
